# almacen-app

Sistema de gestión de almacén desarrollado con Node.js, Express y MySQL.

## Stack
- Frontend: HTML/CSS/JS (SPA)
- Backend: Express.js
- Base de datos: TiDB Cloud (MySQL)
- Deploy: Vercel

## Variables de entorno
Crear un archivo `.env` con DB_HOST, DB_PORT, DB_USER, DB_PASSWORD y DB_NAME.
